Software Engineer
-----------------

Currently, I am the Technical Program Manager for the GIST (Global Information Security & Technology) organization at Bloomreach. I am the sole project manager for all topics related to IT support, infrastructure, security, and security engineering. During my time at Bloomreach, I have completed many notable milestones that include owning Atlassian migrations to the cloud, building the project management office from the ground up, and serving as the liaison between GIST and engineering to drive projects. 

Back in March of 2019 during the start of COVID lockdowns, I discovered a new passion for programming and began taking Udemy courses to learn Python. I have thoroughly enjoyed learning Python and challenging myself to build applications, automation, and other code. After learning the basics through boot camps I became extremely interested in learning more about what Python can do. Since then I have built a portfolio of over 75 projects consisting of web applications, GUI applications, Flask applications, web scraping, automations, and more!
